# AI-Powered iGaming Promotional Marketing Engine

A scalable, AI-powered promotional and marketing automation engine for iGaming platforms, designed to handle 100M+ users.

## Features

- **Event Tracking & Attribution**: Multi-touch attribution with UTM tracking
- **Player Segmentation**: RFM analysis, behavioral, and predictive segmentation
- **Campaign Management**: Email, SMS, Push, WhatsApp campaigns
- **AI-Powered Content**: OpenAI integration for personalized content generation
- **Real-time Personalization**: Sub-200ms recommendation engine
- **Analytics Dashboard**: Comprehensive reporting and insights

## Technology Stack

- **Backend**: Python (FastAPI)
- **Database**: PostgreSQL
- **Cache**: Redis
- **AI/ML**: OpenAI API, scikit-learn, XGBoost
- **Frontend**: HTML, CSS, JavaScript
